# Proyecto_final_eco_de_decision
#Explicación del proyecto
    # Es una experiencia interactiva donde el público responde preguntas y, según sus elecciones, se generan distintos caminos que influyen en el sonido y en el movimiento dentro de un entorno coreográfico
    #El sistema funciona como un árbol de decisiones interactivo donde cada respuesta del usuario activa un nodo distinto.
Cada nodo está asociado a un archivo de audio o a una variación sonora específica.
Dependiendo del camino elegido, el sistema reproduce diferentes pistas, generando múltiples recorridos y finales posibles.

    #¿Cómo funciona? 

           # 1. El usuario recibe una pregunta   
           # 2. Elige una opción           
           # 3. Esa opción activa un sonido específico            
           # 4. El sistema avanza a la siguiente pregunta            
           # 5. Al final, se genera un resultado distinto según el recorrido
